### 📖 What’s the story?

At **Enigma North**, we love Minecraft, but we hate waiting. We think server admins and players should be able to use the best features as soon as a new **Minecraft** version drops—without the long wait for updates that might never come.

### ⚡ What we actually do

When a great plugin gets abandoned or starts lagging on new versions, we step in:
*   **Bringing things back:** We take awesome plugins that stopped updating and make unofficial forks that actually work.
*   **Clean & Fast:** We tweak the code so everything runs smooth on the latest Minecraft releases.
*   **No more waiting:** You don't have to wait months for upstream support. If it's broken, we port it.

### 🌍 100% Open Source

No paywalls, no secrets. **Every single one of our projects is completely open-source.** 
We’re all about the community, so feel free to check the code, report bugs, or help us make things even better.
